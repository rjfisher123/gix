# GIX SDK

Client libraries for external developers to interact with GIX services.

## SDKs

- **rust**: Native Rust SDK (thin wrapper around GIX crates)
- **python**: PyO3 bindings for Python clients
- **js**: WASM-compatible bindings for browser and Node.js
