# GIX Specifications

This directory contains the **source of truth** for all GIX component specifications.

## Structure

- `crypto_spec.md` - Stage 1: Post-quantum cryptography, envelopes, KEM hierarchy
- `gxf_spec.md` - Stage 2: GXF ABI v3 specification
- `ajr/` - AJR (Anonymized Job Routing) specifications
- `gsee/` - GSEE (Secure Execution Envelope) specifications
- `gcam/` - GCAM (Global Compute Auction Mechanism) specifications
- `integrated/` - Cross-component integration specifications

## Development Process

All implementation must be driven by these specifications. If a feature is not specified here, it should not be implemented until the specification is written and reviewed.
